data_reader: Reading and converting data¶

class data_reader.DataReader¶

Implements read/convert operations for csv files

classmethod read_the_data(path_to_file: str | Path | List[str] | List[Path], schema: StructType | None = None, join_df: bool = False) → DataFrame | List[DataFrame]¶

Reads the single or many files and returns them as Spark dataframe or the list of Spark dataframes respectively.

Parameters:

  • path_to_file – path to the initial file or to a list of files

  • schema – Spark schema for the file(s)

  • join_df – if True all the files will be joined in a single DataFrame, otherwise, the list of DataFrames will be returned

Returns:

the Spark dataframe or the list of them

classmethod read_from_dir(path_to_dir: str | Path, schema: StructType | None, limits_of_files: int = -1, join_df: bool = False) → Tuple[DataFrame | List[DataFrame], List[Path]]¶

Reads the files from the directory. Simplify the operation of reading all the files from a particular dir. For some purposes (testing, exploration) only the limited quantity of files can be read.

Parameters:

  • path_to_dir – path to the dir where the files are

  • schema – Spark schema for the file(s)

  • limits_of_files – the parameter defines how many files to read. -1 should be set to read all the files

  • join_df – if True all the files will be joined in a single DataFrame, otherwise, the list of DataFrames will be returned

Returns:

the Spark dataframe or the list of them

classmethod convert_to_parquet(path_to_file: str | Path | List[str] | List[Path], schema: StructType | None, parent_dir_for_save: str | Path, join_df: bool = False) → bool¶

Converts one or several csv files into the parquet format. For using with Spark the parquet format is the faster solution comparing with csv.

Parameters:

  • path_to_file – path to the initial file or to a list of files

  • schema – Spark schema for the file(s)

  • parent_dir_for_save – the directory to save parquet file(s)

  • join_df – if True all the files will be joined in a single DataFrame, otherwise, the list of DataFrames will be returned

Returns:

True, if the operation was successful, False otherwise
